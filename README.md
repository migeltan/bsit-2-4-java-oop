## ⚙️ Software Installation Guide

To ensure everyone is on the same page and our codes run smoothly, we will strictly be using the **Eclipse IDE**. Please follow this guide to set up your device.

### Step 1: Install Java (IMPORTANT)
Before installing Eclipse, you must check if Java is already installed on your device. 
1. Press **Win + R**.
2. Type `cmd` and press Enter.
3. Type `java -version`.
* If Java is not installed, **Download the JDK** from: [https://adoptium.net/](https://adoptium.net/). Install it using the default settings.

### Step 2: Download Eclipse Installer
1. Go to: [https://www.eclipse.org/downloads/](https://www.eclipse.org/downloads/).
2. Click "**Download x86_64**".
3. Download the Eclipse Installer (.exe file).

### Step 3: Run Eclipse Installer
1. Open the downloaded file: `eclipse-inst-jre-win64.exe`.
2. From the menu, select "**Eclipse IDE for Java Developers**".

### Step 4: Choose Installation Folder
* Leave the default location (recommended).
* Click "**Install**".
* Accept the license agreement when prompted.

### Step 5: Launch Eclipse
1. Once installation is complete, click "**Launch**".
2. You will be asked to choose a **Workspace** folder.
   * The default is fine.
   * Click "**Launch**" again to open the IDE.

## 🚀 Repository Setup Guidelines

Follow these steps to get a copy of our class codes onto your local machine. Open your computer's **Terminal**, **Command Prompt**, or **Git Bash** to run these commands.

### Option 1: Clone the Repository (Best for viewing and running)
If you just want to download the code to practice and receive my updates as the semester goes on, clone the repository.

1. Navigate to the folder where you want to save the class files:
   ```bash
   cd C:\Users\Admin\Desktop
2. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/lauurnce/bsit-2-4-java-oop.git
   ```
3. To pull new codes I add in future weeks, navigate inside the folder and run:
   ```bash
   git pull origin main
   ```
### Option 2: Fork the Repository (Best for making your own edits)
If you want your own independent copy of this repository on your GitHub account so you can modify the code, add your own notes, and push changes:

1. Go to the top right of this repository page and click the Fork button.

2. Once the repository is copied to your account, clone your forked version to your local machine:
   ```bash
   git clone [https://github.com/YOUR-USERNAME/](https://github.com/YOUR-USERNAME/)[Repository-Name].git
   ```
